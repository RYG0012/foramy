OLG 模型與 PAYG 制度筆記

一、OLG 模型基本設定

  • 每一代人分為「年輕期」與「老年期」，總共活兩期。
  • 人口每期以固定成長率 n 成長，也就是：
    下一期的人口 = (1 + n) × 本期人口。
  • 年輕人有收入 y，老年時不工作。

二、效用最大化問題

個人希望讓自己一生的效用最大化：

U = u(ct年輕) + β × u(ct+1老年)

條件限制：

  • 年輕期：ct年輕 + st = y（收入要分配到當期消費與儲蓄）
  • 老年期：ct+1老年 = (1 + rt+1) × st（靠年輕時儲蓄及利息過活）

三、最適條件（Euler 方程式）

最佳化會導出：

邊際效用（年輕期） = β × (1 + rt+1) × 邊際效用（老年期）

四、PAYG 制度的加入

政府制度：

  • 每位年輕人繳稅 a
  • 每位老年人領取給付 b
  • 政府不儲蓄，稅收直接發給老人 → 政府預算平衡：
    a × Nt = b × Nt-1
  • 整理得：b = a × (1 + n)

個人預算條件變成：

  • ct年輕 + a + st = y
  • ct+1老年 = b + (1 + rt+1) × st

五、動態一般均衡條件

  • 總資本 Kt 來自所有年輕人的儲蓄 → Kt = St
  • 利率由生產函數決定 → rt = f'(Kt)
  • 資源限制式：

    年輕消費 × Nt + 老年消費 × Nt-1 + 政府支出 ≤ 總產出

七、延伸問題探討

  • 模擬：當人口成長率 n 下降，PAYG 制度會破產嗎？
  • 模擬：改採 Fully-funded（完全資本化）制度能否對抗人口老化？
  • 思考：若儲蓄報酬率 r < n，PAYG 是否仍具永續性？
